# User Stories

1. As a founder, I can add a note about a new idea so I remember the context later.
2. As a founder, I can set a goal with a deadline so I track commitments.
3. As a founder, I can auto-plan the week so goals become actionable tasks.
4. As a founder, I can run a daily reflection to get a concise recap.
5. As a founder, I can search notes to retrieve decisions instantly.
6. As a researcher, I can store Q&A pairs in knowledge and search them offline.
7. As a researcher, I can export plan-week output to share with a teammate.
8. As a busy parent, I can run `make demo` to see output without configuration.
9. As a team lead, I can point `PAIOS_DATA_DIR` to a shared folder to keep state together.
10. As a security-conscious user, I can run without any network calls or secrets.
